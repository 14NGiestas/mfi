  • public pure subroutine crotg(a, b, c, s)

    crotg generates a Givens rotation with real cosine and complex sine:

      [  c         s ] [ a ] = [ r ]
  [ -conjg(s)  c ] [ b ]   [ 0 ]

    where c is real, s is complex, and c**2 + conjg(s)*s = 1.

    Arguments
